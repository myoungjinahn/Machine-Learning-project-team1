# 늦가을 오후 쾌적도 분류기 프로젝트 제안서

---

## 1. 프로젝트 개요

**프로젝트 이름**: 늦가을 오후 쾌적도 분류기
**작성자**: 이희반, 박준호, 안명진
**작성일**: 2026년 4월 5일
**목적**: 11월 오후 시간대의 온도와 미세먼지 수치를 바탕으로 야외활동 적합성을 자동으로 분류하는 모델 생성

### 주요 도구

- **언어**: Python
- **라이브러리**
  - `NumPy` — 행렬 연산
  - `Pandas` - 엑셀 데이터 로드, 최고기온 필터링 및 학습 데이터셋 전처리
  - `scikit-learn` — 데이터 전처리 → 모델링 → 평가
- **주요 알고리즘**: KNN (K-최근접 이웃) — 가까운 데이터끼리는 비슷함 → 유클리드 거리 계산

---

## 2. 배경 및 필요성

아침 예보에서 미세먼지가 '쾌적'하다는 것을 확인하고 마스크 없이 외출했지만, 이내 심한 기침이 시작되었고 미세먼지 농도를 재확인한 결과 실제 농도는 예보와 달리 매우 높았다.

잘못된 데이터가 건강에 직접적인 해를 끼칠 수 있음을 체감한 순간이었으며, 이러한 피해를 방지하고자 **과거 데이터를 바탕으로 미세먼지와 기온을 분석해 외출 적합도를 정확히 판별하는 분류 모델**을 개발하기로 했다.

## 5. 주요 기능 및 특징

### 5-1. 기상 데이터 기반 쾌적도 이진 분류
- 2025년 11월 기상 데이터에서 **일 최고온도**와 **초미세먼지 농도(PM2.5)** 를 입력받아 해당 날씨를 **쾌적(1) / 불쾌적(0)** 으로 자동 분류
- 쾌적 조건 기준:
  - **온도 11°C ~ 18°C** 이고
  - **초미세먼지(PM2.5) 농도 25 이하** 일 시 → **쾌적(1)**
  - 위 조건 중 하나라도 벗어나면 → **불쾌적(0)**

### 5-2. KNN 기반 유사 날씨 탐색
- 새로운 날씨 데이터가 입력되면 기존 학습 데이터 중 **온도와 미세먼지 조건이 가장 유사한 k개의 날**을 찾아 다수결로 쾌적도를 판정
- k값 튜닝을 통해 최적의 이웃 수를 탐색하여 모델 정확도를 향상

### 5-3. 모델 성능 평가
- 아래 4가지 지표로 모델 성능을 다각도로 측정

- **Accuracy** 전체 예측 중 맞은 비율
- **Precision (정밀도)** 쾌적이라고 예측한 것 중 실제로 쾌적한 비율
- **Recall (재현율)** 실제 쾌적한 날 중 모델이 쾌적이라고 맞힌 비율
- **F1-Score** Precision과 Recall의 조화 평균 (불균형 데이터에 유용)
